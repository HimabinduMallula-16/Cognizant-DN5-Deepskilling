# Spring Data JPA & Hibernate

This repository contains hands-on exercises completed as part of the Cognizant Digital Nurture 5.0 Spring Data JPA and Hibernate module.

## Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
- MySQL
- H2 Database
- VS Code

## Projects

### 1. orm-learn
Basic Spring Data JPA CRUD operations.

Topics Covered
- Entity Mapping
- Repository
- Service Layer
- CRUD Operations

---

### 2. querymethodsdemo

Spring Data JPA Query Methods.

Topics Covered

- Derived Query Methods
- Custom Query Methods
- Sorting
- Searching
- Repository Methods

---

### 3. hibernatemappingsdemo

Hibernate Relationship Mapping.

Topics Covered

- One-To-One
- One-To-Many
- Many-To-One
- Many-To-Many
- Entity Relationships

---

### 4. hqljpqldemo

Hibernate Query Language (HQL) & JPQL.

Topics Covered

- HQL
- JPQL
- Native Queries
- Aggregate Functions
- Fetch Join
- Criteria API
