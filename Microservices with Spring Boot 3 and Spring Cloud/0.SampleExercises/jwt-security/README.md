# Exercise 3 – JWT Authentication

## Objective
Implement JWT (JSON Web Token) based authentication in a Spring Boot application.

## Technologies
- Java 17
- Spring Boot 3.5.16
- Spring Security
- JJWT 0.12.7
- Maven

## Features
- JWT Token Generation
- Secure REST API
- Spring Security Configuration

## Endpoint

GET /token

Returns a generated JWT token.

## Run

```bash
.\mvnw.cmd spring-boot:run
```

Application URL:

```
http://localhost:8082/token
```

## Expected Output

A valid JWT token is generated and returned.
